Olá!

Esta é a página onde pode encontrar a sua lista das compras. Esta lista é gerada segundo uma seleção de alimentos essenciais que devem estar no seu frigorífico e/ou na sua despensa.

A lista representa os alimentos considerados em falta, podendo estes serem removidos. Também é possível adicionar alimentos.

Elementos da Página

Alimentos:
Alimentos possíveis para escolha.
Lista das Compras:
Lista de alimentos escolhidos para comprar, posteriormente.

Botões

:
Adicionar o alimento selecionado à lista das compras.
:
Fazer a encomenda online da lista das compras.
